# INF231-PMP-Project
Projet académique - INF231 : Coordinateur : Pr Melatagia - Université de Yaoundé 1 Année académique : 2025-2026 🎯 Description Application de traitement d'images en langage C supportant le format PPM (Portable PixMap). Le programme offre une interface interactive pour manipuler des images avec diverses opérations de traitement.
